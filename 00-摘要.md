高校校园中遗失与拾获物品在宿舍、教学楼与公共区域频繁出现，公告栏与即时群聊仍是主要发布渠道，信息分散且时效受限，使失主与拾获人难以及时形成有效匹配。现有流程缺乏统一的身份核验与证据留存，物品状态、认领记录与通知链条难以追溯，管理端也缺少对全局数据的统计、监督与治理能力。针对上述问题，本课题设计并实现面向高校场景的失物招领系统，采用前后端分离架构，将失物发布、招领展示、认领申请与状态流转纳入统一业务链路。前端基于 Vue 3、Vite 与 Element Plus 构建交互界面，提供物品分类检索、时间地点过滤、详情展示、图片浏览与个人中心管理等能力，提升信息获取效率与操作一致性。后端使用 Spring Boot 3.2、MyBatis-Plus、Spring Security、JWT 与 MySQL 构建用户权限、物品信息、分类与状态、认领申请、通知消息及文件资源等模块，支持图片上传、审核流转与消息提醒。系统在保持操作简洁性的同时强化数据可追溯与审计价值，为校园失物管理提供规范化、可管理的服务支撑与信息化基础。关键词：失物招领；校园服务；Spring Boot；Vue 3
