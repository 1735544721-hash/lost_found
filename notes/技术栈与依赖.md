# 技术栈与依赖

## 后端
- Java 17（`pom.xml`）
- Spring Boot 3.2.0
- Spring Web / Spring Security
- MyBatis-Plus 3.5.7
- MySQL Connector/J
- Knife4j OpenAPI3
- Hutool 5.8.25
- JWT（`java-jwt` 4.4.0）
- Alipay SDK 4.39.185.ALL

## 运行与配置要点
- 服务端口：`server.port=1235`
- 数据源：MySQL（`lost_found_db`）
- OpenAPI/Swagger：`/swagger-ui.html`
- Knife4j Basic：`admin/admin`
- 前端代理：`/api` -> `http://localhost:1235`

## 前端
- Vue 3 + Vite
- Vue Router / Vuex / Pinia
- Element Plus + Icons
- Axios
- Font Awesome
