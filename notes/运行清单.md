# 运行清单

## 初始化数据库
- 在 MySQL 中创建数据库 `lost_found_db`
- 导入 `失物招领系统/lost_found_db.sql`

## 后端启动
- 进入 `失物招领系统/springboot`
- 按需更新 `src/main/resources/application.properties` 中的 `spring.datasource.*`
- 运行 `./mvnw spring-boot:run`（或 `mvn spring-boot:run`）
- 服务地址：`http://localhost:1235/`
- 接口文档：`http://localhost:1235/swagger-ui.html`
- Knife4j 基础认证：`admin/admin`

## 前端启动
- 进入 `失物招领系统/vue3`
- 安装依赖：`npm install`
- 启动开发环境：`npm run dev`
- 默认地址：`http://localhost:3002/`
- 前端代理：`/api` -> `http://localhost:1235`
