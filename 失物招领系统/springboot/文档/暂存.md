1.JDK（Java开发环境）：系统后端基于 Java 语言开发。本项目 `pom.xml` 指定 Java 版本为 17，需安装 JDK 17 作为编译、运行与调试环境支撑，并满足 Spring Boot 3.2.0 的运行要求。
2.IntelliJ IDEA 2019：后端项目主要开发工具（IDE），用于 Spring Boot 工程的代码编写、依赖管理、调试与运行等。
3.Visual Studio Code：前端项目开发工具（如配套 Vue 前端），用于页面编写、前端调试与项目构建，并可用于与后端接口联调。
4.MySQL 8.0.19：关系型数据库管理系统，用于保存用户信息、物品分类、失物信息、招领信息、认领申请、通知消息等核心业务数据，作为系统数据持久化的基础支撑（见 `application.properties` 中 `lost_found_db` 配置）。
5.Tomcat（Servlet容器）：用于部署后端 Java Web 服务。Spring Boot 默认内嵌 Tomcat，可直接作为系统 Web 服务容器运行。
6.Nginx 1.14.8：用于 Web 服务的反向代理与静态资源转发等。在系统部署阶段可作为统一入口，提升访问稳定性与可维护性。
7.SpringBoot：后端核心框架。本项目基于 Spring Boot 3.2.0 构建，用于快速搭建 Web 应用与接口服务，提高开发效率与工程化水平。
8.MyBatis：持久层框架，用于完成后端对 MySQL 数据库的访问与数据映射。本项目采用 MyBatis-Plus（MyBatis 增强框架）实现分页查询与对象映射，提升数据操作的规范性与开发效率。
9.Vue.js：前端框架（若建设完整前端界面），用于实现系统页面交互、组件化开发与数据展示，支撑失物/招领发布、认领审核、消息通知与统计展示等界面功能。
